

                                  !!!
Remember to remove previous version of Cruel Hessian:

- type 'make uninstall' as root in source directory of old version
OR
- delete /usr/share/cruelhessian/ directory and /usr/bin/cruelhessian file

and delete config file (~/.cruelhessian/options.ini) before installing a new version.
New config file will be placed in ~/.config/cruelhessian directory.
                                  !!!



Building Cruel Hessian with CMake
------------------------

Type :

   * cmake .
   * make
   
   	and (as root)
   	
   * make install

in the main directory.
